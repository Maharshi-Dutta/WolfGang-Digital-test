# Wolfgang Digital — Awards Page

A simplified recreation of the Wolfgang Digital Awards page, built as a React technical assessment.

## How to Run

Make sure you have Node.js installed, then run:

npm install

npm run dev

Open your browser at http://localhost:3000

## Libraries and Tools Used

- React 19

- TypeScript

- TanStack Start (SSR-capable React framework with file-based routing)

- TanStack Router

- Tailwind CSS v4

- shadcn/ui (Radix UI component primitives)

- Embla Carousel (for the insights carousel)

- Lucide React (icons)

- Vite (build tool)

## Features

- Awards filtered by group (e.g. Global Search Awards) across both 2024 and 2025

- Hover interaction on award rows showing a floating image that follows the cursor

- Scroll-aware sticky header that hides on scroll down and reappears on scroll up

- Responsive layout with full-screen mobile navigation

- Static hardcoded data — no backend or API required
